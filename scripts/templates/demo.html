SAXI Icon Font Demo

This demo showcases the SAXI Icon Font with various styles and usage examples.

CSS Class Usage
Use CSS classes to display icons. Follow the FontAwesome-style convention. 
<i class="saxi-solid saxi-archive-add"></i>
Ligature Usage
Use ligatures to display icons by simply typing the icon name within the element. 
<i class="saxi-solid">archive-add</i>

Design Tool Integration
To use these icons in design tools like Figma or Canva:
  1. Install the font files (.ttf) on your system
  2. Select text in your design tool and set the font to:
    • saxi-icons-pro (with font weight 700 for Bold, 400 for Linear, 300 for Outline)
    • saxi-icons-pro-duotone for Duotone icons
    • saxi-icons-pro-broken for Broken icons
  3. Type the icon name (e.g., archive-add) directly to use the ligature
  4. Alternatively, paste the Unicode character directly
